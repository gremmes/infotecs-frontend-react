# Тестовое задание для стажера React Frontend-разработчика
## Стек технологий:
- TypeScript
- Webpack + Npm
- React 16
- Ant design 5 и styled-components
- React router 6
- TanStack Query 4 + axios
- dayjs
- Git (github.com)
- Архитектура FSD (Feature-Sliced Design)
## Требования:
1. Использовать api запросы, созданные с помощью бесплатного ресурса: mockapi.io
2. Проект должен разрабатываться в публичном github репозитории, для будущей
проверки.
3. Для разработки использовать только список стека технологий
## Что было сделано:
- Реализована страница авторизации с формой входа, валидацией и мок-авторизацией
- Создана страница 404 с сообщением "Страница не найдена" и навигацией
- Реализована страница пользователей с таблицей и кнопкой для создания нового пользователя
- Сделаны модальные окна для редактирования и создания пользователя, с формами и сохранением данных
- Использована архитектура Feature-Sliced Design для структуры проекта
- Интерфейс стилизован с помощью styled-components и Ant Design
- Настроена маршрутизация с React Router 6
- Обеспечена работа с данными через TanStack Query и Axios (мок API)
